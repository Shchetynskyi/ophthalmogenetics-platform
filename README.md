# Офтальмогенетична платформа

Візуальний MVP сайту для обговорення структури платформи супроводу пацієнтів із генетичними захворюваннями сітківки.

## Live demo

https://ophthalmogenetics-platform.vercel.app

## Що є в MVP

- Кабінет родини
- Картка пацієнта
- Стрічка супроводу
- Документи
- Дослідження і лікування
- Обмін документами
- Листи до клінік
- Кабінет лікаря
- Черга оцінок лікаря
- Адмін-вид сутностей

## Технічно

React + Vite + TypeScript + Tailwind.

Без бази даних, авторизації, платежів, API та медичної production-логіки.

## Локальний запуск

npm install
npm run dev

## Production build

npm run build
